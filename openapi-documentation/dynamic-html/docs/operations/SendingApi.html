This is the API

sendAdvMail

Sends An email through one of your mail orders allowing additional options such as file attachments, cc, bcc, etc.

URL

https://api.mailbaby.net/mail/advsend

HTTP Method

POST

Response Type

GenericResponse

Parameters

  • Form: subject String(string)

    The subject or title of the email

  • Form: body String(string)

    The main email contents.

  • Form: from EmailAddressName(EmailAddressName)

  • Form: to List(EmailAddressName)

    A list of destionation email addresses to send this to

  • Form: replyto List(EmailAddressName)

    (optional) A list of email addresses that specify where replies to the email should be sent instead of the _from_ address.

  • Form: cc List(EmailAddressName)

    (optional) A list of email addresses to carbon copy this message to. They are listed on the email and anyone getting the email can see this full list of Contacts who received the email as well.

  • Form: bcc List(EmailAddressName)

    (optional) list of email addresses that should receive copies of the email. They are hidden on the email and anyone gettitng the email would not see the other people getting the email in this list.

  • Form: attachments List(MailAttachment)

    (optional) File attachments to include in the email. The file contents must be base64 encoded!

  • Form: id Long(long)

    (optional) ID of the Mail order within our system to use as the Mail Account.

sendMail

Sends an email through one of your mail orders. *Note*: If you want to send to multiple recipients or use file attachments use the advsend (Advanced Send) call instead.

URL

https://api.mailbaby.net/mail/send

HTTP Method

POST

Response Type

GenericResponse

Parameters

  • Form: to String(string)

    The Contact whom is the primary recipient of this email.

  • Form: from String(string)

    The contact whom is the this email is from.

  • Form: subject String(string)

    The subject or title of the email

  • Form: body String(string)

    The main email contents.
