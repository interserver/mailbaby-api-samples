sendMail

Sends an email through one of your mail orders. *Note*: If you want to send to multiple recipients or use file attachments use the advsend (Advanced Send) call instead.

URL

https://api.mailbaby.net/mail/send

HTTP Method

POST

Response Type

GenericResponse

Parameters

  • Form: to String(string)

    The Contact whom is the primary recipient of this email.

  • Form: from String(string)

    The contact whom is the this email is from.

  • Form: subject String(string)

    The subject or title of the email

  • Form: body String(string)

    The main email contents.

viewMailLog

Get a listing of the emails sent through this system

URL

https://api.mailbaby.net/mail/log

HTTP Method

GET

Response Type

MailLog

Parameters

  • Query: id Long

    The ID of your mail order this will be sent through.

  • Query: origin String

    originating ip address sending mail

  • Query: mx String

    mx record mail was sent to

  • Query: from String

    from email address

  • Query: to String

    to/destination email address

  • Query: subject String

    subject containing this string

  • Query: mailid String

    mail id

  • Query: skip Integer

    number of records to skip for pagination

  • Query: limit Integer

    maximum number of records to return

  • Query: startDate Long

    earliest date to get emails in unix timestamp format

  • Query: endDate Long

    earliest date to get emails in unix timestamp format
