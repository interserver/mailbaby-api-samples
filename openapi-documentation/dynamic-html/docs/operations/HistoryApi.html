This is the API

getStats

Returns information about the usage on your mail accounts.

URL

https://api.mailbaby.net/mail/stats

HTTP Method

GET

Response Type

MailStatsType

Parameters

  • Query: time String

    The timeframe for the statistics.

viewMailLog

Get a listing of the emails sent through this system

URL

https://api.mailbaby.net/mail/log

HTTP Method

GET

Response Type

MailLog

Parameters

  • Query: id Long

    The ID of your mail order this will be sent through.

  • Query: origin String

    originating ip address sending mail

  • Query: mx String

    mx record mail was sent to

  • Query: from String

    from email address

  • Query: to String

    to/destination email address

  • Query: subject String

    subject containing this string

  • Query: mailid String

    mail id

  • Query: skip Integer

    number of records to skip for pagination

  • Query: limit Integer

    maximum number of records to return

  • Query: startDate Long

    earliest date to get emails in unix timestamp format

  • Query: endDate Long

    earliest date to get emails in unix timestamp format

  • Query: replyto String

    Reply-To Email Address

  • Query: headerfrom String

    Header From Email Address

  • Query: delivered String

    Limiting the emails to wether or not they were delivered.
