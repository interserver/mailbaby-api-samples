MailBaby Email Delivery API

Send emails fast and with confidence through our easy to use REST API interface.

üìå Overview

This is the API interface to the Mail Baby Mail services provided by InterServer. To use this service you must have an account with us at my.interserver.net.

üîê Authentication

In order to use most of the API calls you must pass credentials from the my.interserver.net site.

We support several different authentication methods but the preferred method is to use the API Key which you can get from the Account Security page.

array[SendMailAdv_replyto_inner] (optional) A list of email addresses that specify where replies to the email should be sent instead of the from address.

array[SendMailAdv_cc_inner] (optional) A list of email addresses to carbon copy this message to. They are listed on the email and anyone getting the email can see this full list of Contacts who received the email as well.
bcc (optional)
array[SendMailAdv_bcc_inner] (optional) list of email addresses that should receive copies of the email. They are hidden on the email and anyone gettitng the email would not see the other people getting the email in this list.

array[SendMailAdv_attachments_inner] (optional) File attachments to include in the email. The file contents must be base64 encoded!
